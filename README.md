# Feature Engineering — Implementations in Python

This repository contains the notebooks developed for the **Feature Engineering** course.  
It includes techniques for data transformation, encoding, and preprocessing to improve the performance of Machine Learning models.

Each topic is implemented in a separate Jupyter Notebook (`.ipynb`).

---

## Contents

* `Codificación con base en la media.ipynb` — Mean target encoding  
* `Compensación de sesgos positivos.ipynb` — Techniques to correct positively skewed distributions  
* `Conteo de categorías.ipynb` — Frequency/count encoding  
* `Creación de variables binarias.ipynb` — One-hot encoding and dummy variables  
* `Datos atípicos.ipynb` — Outlier detection and treatment  
* `Distribución de datos.ipynb` — Data distribution analysis and normality  
* `Feature Hashing.ipynb` — Hashing technique for high-cardinality categorical variables  
* `Relaciones lineales.ipynb` — Analysis of linear relationships between variables  
* `Transformaciones de potencia.ipynb` — Box-Cox and other power transformations  
* `Transformaciones lineales.ipynb` — Scaling and normalization techniques  
* `Tratamiento de datos faltantes.ipynb` — Missing data imputation  
* `Variables categóricas ordinales.ipynb` — Encoding for ordinal categorical variables  

---

## Objective

Apply feature engineering techniques to properly prepare datasets before modeling, improving predictive performance and model stability.
